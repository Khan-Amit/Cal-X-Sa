## 📊 What is Cal-Ex?

Cal-Ex (Calculation Exchange) is a **zero-dependency, offline-first** tax intelligence platform providing:

- Real-time progressive tax calculation for 4+ countries
- Embedded tax brackets (no API calls — ever)
- Import duty & VAT lookup
- Clean, bottle-green interface
